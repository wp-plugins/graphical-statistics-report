This plug-in will display the graphical report for admin about, 

   1. Monthly user registration count graphical report,
   2. Monthly post count graphical report,
   3. Monthly comments posted count graphical report
   4. Category wise post count graphical report.

**To see the report:**

1. Activate the plugin.
2. Go to 'Graphical statistics' link under SETTINGS TAB to see the report graph.  

Thanks to fusion Charts team for this SWF graph.

== Installation ==

**Installation Instruction & Configuration**  

*   Unpack the *.zip file and extract the /graphical-statistics-report/ folder.    
*   Drop the 'graphical-statistics-report' folder into your 'wp-content/plugins' folder    
*   In word press administration panels, click on plug-in from the menu.    
*   You should see your new 'graphical statistics report' plug-in listed under Inactive plug-in tab.    
*   To turn the word presses plug-in on, click activate.    
*   Go to Graphical statistics link under settings to see the report graph.
